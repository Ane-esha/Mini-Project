# Mini-Project
This project analyzes Disney+ Movies and TV Shows using Power BI. After cleaning the dataset in Excel, a three-page interactive dashboard was created. Users can search titles, filter content by genre, type, year, rating, country, and continent, and explore the distribution of movies and TV shows through interactive visualizations and insights.

Collected the Disney+ Movies and TV Shows dataset.(- <a href="https://www.kaggle.com/datasets/shivamb/disney-movies-and-tv-shows">Dataset view</a)

Cleaned and preprocessed the data in Excel by handling missing values and correcting data formats.

Imported the cleaned dataset into Power BI.

Created additional tables and columns required for analysis.

Built relationships between tables and performed data modeling.

Designed a three-page interactive dashboard.

Added filters and search options for genre, type, year, rating, country, and continent.

Created visualizations to analyze the distribution of movies and TV shows.

Implemented navigation and reset filter functionality for better user experience.
